# 🌶️ SpiceKit

SpiceKit は、ゲーム開発向けの再利用可能な C# ライブラリやツールを集めたコレクションです。

軽量で実践的、かつ依存関係の少ないコンポーネントを提供し、さまざまなゲームプロジェクトへ簡単に導入できることを目指しています。各リポジトリは特定の課題に焦点を当てており、単体でも、他の SpiceKit プロジェクトと組み合わせても利用できます。

## コンセプト

SpiceKit は、以下の考え方を大切にしています。

* 軽量で理解しやすいこと
* 外部ライブラリへの依存を最小限に抑えること
* 実際のゲーム開発で役立つこと
* モジュール化され再利用しやすいこと
* オープンソースとして公開すること

大規模な統合フレームワークを提供するのではなく、必要な機能だけを選んで導入できる、小さく独立したライブラリ群を目指しています。
